# Local AI Service API

## Goal

Define the contract between the Flutter app and the local indexing / inference
service.

## Runtime

- Local process on the same machine
- Recommended first implementation: Python + FastAPI
- Transport: HTTP on `127.0.0.1`

## Base URL

`http://127.0.0.1:4318`

## Endpoints

### `GET /health`

Returns service availability and model readiness.

Example response:

```json
{
  "status": "ok",
  "models": {
    "clip": "ready",
    "face": "ready"
  }
}
```

### `GET /sources`

Returns configured media sources.

### `POST /sources`

Create a source.

Request:

```json
{
  "source_type": "local_folder",
  "display_name": "Mac Photos Export",
  "root_path": "/Users/john/Pictures/Exports"
}
```

### `POST /scan-jobs`

Create a scan task for a configured source.

Request:

```json
{
  "source_id": "source_001",
  "recursive": true,
  "mode": "incremental"
}
```

Response:

```json
{
  "job_id": "scan_20260318_001",
  "status": "completed"
}
```

### `GET /scan-jobs/{job_id}`

Returns progress and counters.

### `GET /dashboard`

Returns the aggregate data required by the home dashboard.

Suggested payload sections:

- `stats`
- `sources`
- `smart_albums`
- `signals`
- `recent_events`
- `scan_jobs`

### `GET /albums`

Returns smart albums generated by the service.

Query params:

- `type`
- `sort`
- `source_id`
- `limit`

### `GET /assets`

Returns scanned media assets.

Query params:

- `album_type`
- `source_id`
- `tag`
- `limit`

### `GET /people`

Returns clustered people and review states.

### `POST /people/{cluster_id}/confirm`

Assign a human identity to a cluster.

Request:

```json
{
  "name": "我",
  "is_self": true
}
```

### `GET /timeline`

Returns event-like memory groups for the timeline screen.

### `POST /corrections`

Stores user feedback on labels, people, or duplicate suggestions.

Request:

```json
{
  "asset_id": "asset_001",
  "kind": "album_label",
  "from": "travel",
  "to": "daily"
}
```
