# UI Direction

## References

Visual direction for this app is based on:

- Horizon UI Chakra GitHub repository: https://github.com/horizon-ui/horizon-ui-chakra
- Apple-style desktop and mobile productivity apps

## Visual principles

### Calm premium surfaces

- Soft neutral backgrounds instead of flat white
- Large-radius cards
- Layered translucency and depth, used sparingly

### Data-rich but not noisy

- Dashboard information grouped in elegant cards
- High visual hierarchy through spacing, typography, and contrast
- Metrics framed as memory signals, not admin counters

### Fashion-forward feel

- Refined sans serif body type
- Distinct display typography for hero sections
- Gentle gradients, deep navy anchors, and controlled accent blues

### Product personality

This app should feel like:

- A private gallery
- A memory studio
- A beautiful intelligence layer on top of a NAS

It should not feel like:

- A backup utility
- A generic file manager
- A developer dashboard

## Translation into Flutter

The first Flutter UI uses:

- A luminous background with layered radial gradients
- Rounded glass cards with subtle borders
- Compact stat chips and elevated section headers
- A split layout that expands elegantly on larger screens
