# Roadmap

## Phase 1: Multi-source indexing foundation

- Generic source management
- Local folder and SMB-mounted path scanning
- SQLite media indexing
- Global asset library
- Smart albums
- Manual album correction
- Open original file from indexed asset

## Phase 2: Better organization and retrieval

- Real image thumbnails and video keyframes
- EXIF extraction and source-aware metadata
- Better event grouping by day and source context
- Tag hierarchy and user-defined tags
- People confirmation
- Duplicate detection
- Better search and filtering
- Incremental indexing

## Phase 3: Memory intelligence

- Face clustering pipeline
- Embedding-based semantic search
- Cross-source grouping and deduplication
- Timeline memory cards
- Hybrid local/cloud enrichment
- iPhone and Android polish
